#  Парсер НСИ для ERP: Насосное оборудование

**Инструмент автоматизированного сбора и актуализации нормативно-справочной информации (НСИ) о насосном оборудовании с сайтов поставщиков.**

Проект предназначен для парсинга технических характеристик, цен и остатков оборудования для последующей выгрузки в ERP-систему (например, 1С, SAP, Oracle).

---

## 🌐 Источники данных

Парсер настроен на работу со следующими источниками:

1. **[nasoscentr.ru](https://nasoscentr.ru/)** (Насос-Центр)
   - *Структура:* Bitrix + Aspro.Max
   - *Данные:* Каталог насосов (Д, 1Д, 2Д и др.), цены, характеристики (Таблица `wd_propsorter`).
2. **[prm.elcomspb.ru](https://prm.elcomspb.ru/)** (ГК Элком)
   - *Структура:* Кастомный дизайн (Bitrix)
   - *Данные:* Широкий ассортимент (Насосы, Электродвигатели), детальные характеристики (Блок `chars`), наличие на складах.

---

## ✨ Возможности

- ✅ **Автоматический сбор:** Извлечение номенклатуры, артикулов, цен и технических характеристик.
- ✅ **Адаптивность:** Поддержка разных HTML-структур (Битрикс, кастомные шаблоны).
- ✅ **Фильтрация:** Возможность сбора только определенного типа оборудования (например, только "Консольные насосы").
- ✅ **Валидация:** Проверка дубликатов и корректности числовых данных (подача, напор).
- ✅ **Экспорт:** Выгрузка собранных данных в JSON/CSV для интеграции с ERP.

---

## 📂 Структура извлекаемых данных

На основе анализа HTML-кода страниц товаров, парсер извлекает следующие поля:

| Поле (Ключ) | Описание | Пример значения | Источник |
|-------------|----------|-----------------|----------|
| `model` | Наименование товара | "Насос Д 160-112м" | `h1`, `.product-title` |
| `article` | Артикул / Код товара | "2114", "02.01.216902" | `.sku`, `.product_item__top-art` |
| `price` | Цена | "368 782 руб" | `.price`, `.thisPrice` |
| `brand` | Производитель | "ГМС Ливгидромаш", "ESQ" | `.properties`, `.chars__row` |
| `flow_rate` | Подача | "160, 90 м³/ч" | Таблица характеристик |
| `head` | Напор | "122, 30 м" | Таблица характеристик |
| `power` | Мощность двигателя | "110, 18.5 кВт" | Таблица характеристик |
| `speed` | Частота вращения | "2900, 1450 об/мин" | Таблица характеристик |
| `availability` | Наличие | "Наличие уточняйте", "2 шт" | `.item-stock` |

## 🛠️ Установка и запуск

### 1. Требования
- Python 3.8+
- Менеджер пакетов pip

### 2. Установка зависимостей
Создайте виртуальное окружение и установите необходимые библиотеки (requests, beautifulsoup4, pandas и др.):

```bash
# Создание виртуального окружения
python -m venv venv

# Активация (Linux/Mac)
source venv/bin/activate

# Активация (Windows)
venv\Scripts\activate

# Установка пакетов
pip install -r requirements.txt
```

### 3. Запуск парсера
Запустите основной скрипт проекта. По умолчанию парсер может работать в режиме сбора всех данных:
```
python main.py
```
